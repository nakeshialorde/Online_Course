Representation of a Set

Description

A set is always denoted by a capital letter. For example: the set A or the set B. The elements are denoted by lower case letters or numbers and are enclosed by curly brackets {a,b,c} or {1,2,3}. The order of the elements in a set is insignificant, but are not repeated by set notation.

The elements of a set can be represented by the following numbers:
  1. Natural Numbers: A={1,2,3,4}.
  2. Rational Numbers: A={1⁄2,1⁄3,1⁄4}.
  3. Real Numbers: A={-∞,...,∞}.
  4. Prime Numbers:A={2,5,7,11,13}.
  5. Integers:A={...,-1,0,1,2}.
  6. Even Integers:A={2,4,6,8}.
  7. Odd Integers:A={5,7,9,11}.
